# catppuccin-dotfiles

Personal dotfiles themed with [Catppuccin](https://catppuccin.com/), configured for a minimal Wayland setup using Hyprland.

Includes configurations for:
- Hyprland
- Zsh (with Oh My Zsh)
- Waybar
- Rofi
- Mako
- Alacritty
- GTK themes
- Wlogout
